# investigate-incident evals

Evaluates `investigate-incident` in standalone operation. Checks whether it can organize hypotheses and next steps for an unknown-cause incident without making production changes.

## Iter 0

- `description` and body are aligned on "unknown-cause incident investigation"
- Hypotheses are separated from confirmed facts
- Next step is narrowed to one confirmable action
- Production changes are never suggested during investigation

## Open items
